# The Clone
The Youtube clone is a fairly simple project. It showcases the framework of HTML and CSS. It was a fun project and I can't wait for my next adventure!

The project started out with building out the base design and struckering the vidoes. The videos themselves where all wrapped in their each individal div, and then each video was wrapped in one single div, the section element. 

The section then was converted into a CSS grid using columns with the 1fr.

After the videos where properly designed, the header and the sidebar(nav) needed to be designed. This process was used with Flexbox. Using key CSS properties like, "justy-content" and "align items".

Overall this project was amazing! And anyone who is newer to web developing should definitly give this a go and even try Facebook, or Twitter. Even Reddit!
